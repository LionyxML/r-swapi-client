Esse é um cliente para a Star Wars API (swapi).

Procure a personagem desejada digitando o nome ou parte do nome na barra de pesquisa. O resultado é mostrado de forma assíncrona (pelo fetch()).

Escolha a forma de saída desejada entre: cards (algumas características traduzidas), Console (apenas os nomes) e Console Completo (o objeto completo).
